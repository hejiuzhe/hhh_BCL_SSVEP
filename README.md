# SSVEP 脑控无人机系统
本项目旨在设计和实现一个基于稳态视觉诱发电位（SSVEP）的脑机接口（BCI）系统，用于实时、高精度地控制无人机进行飞行。我们采用经典的 FBCCA（Filter Bank Canonical Correlation Analysis）算法作为实时控制基线，并探索深度学习模型（如 CNN/Transformer）以进一步优化分类性能。

## ✨ 核心特性
- 实时 SSVEP 分类：采用 FBCCA 算法实现高准确率和高信息传输率（ITR）的指令识别。
- 多指令控制集：支持 5 个核心指令（8Hz, 9Hz, 10Hz, 11Hz, 13Hz）的切换，用于控制无人机的起飞、降落和运动。
- Python 驱动：算法和控制逻辑完全基于 Python 语言实现，易于集成和部署。
- 硬件兼容性：兼容 $\ge 8$ 通道、采样率 $\ge 250$ Hz 的科研级入门脑电设备。
- 闭环飞行控制：实现算法输出到无人机 SDK 指令发送的完整闭环。

## 🛠️ 核心算法
| 算法名称               | 简称   | 特点                                   | 角色                                       |
| ---------------------- | ------ | -------------------------------------- | ------------------------------------------ |
| Filter Bank CCA        | FBCCA  | 免训练，高实时性，抗噪能力强           | 实时控制基线 (Baseline)，用于稳定控制无人机 |
| 标准 CCA               | CCA    | 免训练，计算速度极快                   | 算法流程验证和初步测试                     |
| 深度学习模型（CNN/EEGNet） | -      | 需要大量数据训练，分类精度潜力高       | 后期优化和论文对比分析                     |

## 📐 指令频率映射
系统采用 1 秒滑动窗口进行实时识别，输出结果映射为无人机指令：

| 刺激频率 | 映射动作                              |
| -------- | ------------------------------------- |
| 8 Hz     | 前进 / 起飞                           |
| 9 Hz     | 后退 / 降落                           |
| 10 Hz    | 左转                                  |
| 11 Hz    | 右转                                  |
| 13 Hz    | 模式切换 (例如：从前进模式切换到旋转模式) |
